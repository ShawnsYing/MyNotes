emacs快捷键功能对照表及壁纸

发表于 2024-04-27 分类于 emacs

多次尝试入坑emacs失败，但是他的快捷键确实很实用，在vscode上装了emacs的快捷键插件，

不过，虽然控制光标用的得心应手，却总是忘记复制粘贴怎么按

整理了一下这些快捷键的中文翻译，放在桌面上随时查阅

自己使用，比较粗糙，基本是机翻，看得懂就行

对照表

移动命令
C-f 向前迈进
C-b 向后移动
C-n 移动到下一行
C-p 移动到上一行
C-a 移动到行首
C-e 移动到行尾
M-f 向前移动一个单词单位
M-b 向后移动一个单词单位
M-> 移动到缓冲区的末尾
M-< 移动到缓冲区的开头
C-v 向下滚动一个屏幕单位
M-V 向上滚动一个屏幕单元
M-g g 跳转到行（命令皮肤）
M-g n 跳转到下一个错误
M-g p 跳转到上一个错误
C-l 跳转至当前行中心
搜索命令
C-s 向前搜索
C-r 向后搜索
A-% 替换
C-Enter 替换一个匹配项（在替换对话框中）
C-M-n 将所选内容添加到下一个查找匹配项
编辑命令
C-d 向右删除 （DEL）
C-h 向左删除 （BACKSPACE）
M-d 向右删除单词
M-Bksp 向左删除单词
C-k 删到句尾
C-S-BKSP 删除整行
C-o 在另一个窗格打开文件但不移动光标
C-w 剪切
M-w 复制
C-y 粘贴
C-j 输入
C-m 输入
C-x C-o 删除周围的空行
C-x h 全选
C-x u （C-/， C-_） 撤消
C-; 切换行注释
M-; 切换区域注释
C-x C-l 转换为小写
C-x C-u 转换为大写
文档命令
C-x C-s 保存
C-x C-w 另存为
C-x C-n 打开新窗口

|选项卡操作命令||
|—|—|
|C-x b |切换到另一个打开的缓冲区|
|C-x C-f| 快速打开文档|
|C-x k |关闭当前选项卡（缓冲区）|
|C-x C-k| 关闭所有选项卡|
|C-x 0 |关闭当前组中的编辑器。|
|C-x 1 |关闭其他（拆分）组中的编辑器。|
|C-x 2 |水平拆分编辑器|
|C-x 3 |垂直拆分编辑器|
|C-x 4 |切换拆分布局（垂直到水平）|
|C-x o |切换其他拆分编辑器|

|其他命令||
|—|—|
|C-g |取消|
|C-space |设置标记|
|C-quote| IntelliSense 建议|
|M-x |打开命令皮肤|
|C-M-SPC |Toggle 侧边栏可见性|
